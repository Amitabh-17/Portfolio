# My Portfolio (B.Tech CSE - Data Science)

This is my personal portfolio website built with **Next.js + TypeScript + Tailwind CSS**.
It uses a modern futuristic theme with smooth scrolling, animated sections, and interactive UI.

## About Me
I am a **3rd-year B.Tech CSE student (Data Science specialization)**.
I am currently focused on:
- Data Science fundamentals
- Machine Learning basics
- Full-stack web development
- DSA and problem solving

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js / React Three Fiber
- shadcn/ui components

## Features
- Smooth one-page portfolio navigation
- Hero section with 3D scene
- About section with learning timeline
- Project showcase with filtering
- Skills/services presentation
- Contact section with interactive UI
- Dark/Light theme toggle

## Run Locally
```bash
npm install
npm run dev
```
Open: `http://localhost:3000`

## Build
```bash
npm run build
npm run start
```
